Your coding sessions
deserve a companion

buddy-evolution is a Claude Code plugin that turns every session into progress. Earn achievements, grow stats, track streaks, and build a journal of your development journey.

// features

Everything runs locally.
Nothing leaves your machine.

🏆

34 Achievements

Across 6 categories — Coding, Testing, Debugging, Consistency, Exploration, and Meta. Including 2 hidden ones to discover.

📈

XP & Leveling

20 levels with evolution tiers. Earn XP from tool calls, file edits, test runs, and session streaks up to 2x multiplier.

📓

Session Journal

Automatic monthly logs with metrics, achievements, and milestones. A personal coding diary that writes itself.

🧠

Stat Growth

DEBUGGING, PATIENCE, CHAOS, WISDOM, SNARK — stats that grow from your activity patterns with diminishing returns.

📂

File Familiarity

Your buddy learns which files and projects you work on most. New → Familiar → Expert → Nostalgic.

⚡

Zero Dependencies

Pure Node.js built-in modules. No npm install. No build step. No telemetry. Just clone and go.

// install

One command. Zero setup.

# Clone into your plugins directory $ git clone https://github.com/FrankFMY/buddy-evolution ~/.claude/plugins/buddy-evolution # Restart Claude Code — your buddy appears on next session

No npm install. No build step. No configuration. Works immediately.

// specification

Built from the community spec

This plugin implements the buddy-evolution specification — a collaborative design born from GitHub issues #41684 and #41867 in the Claude Code repository. The spec covers achievement-based progression, stat synergies, session journals, and file familiarity tracking.
